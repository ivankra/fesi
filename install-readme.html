FESI - Free EcmaScript Interpreter
==================================


The FESI installer will create the following directory structure:


	c:/program files/fesi - (or whaterver your target directory is)
	   /lib - contains fesi.jar
	   /bin – contains launcher command files
	   /doc/html – Contains doc in html format
	   /doc/html/api – Contais javadoc of the Jslib (only)

	
         /examples – contains example code, if the pack 'examples' is loaded
	
	if the pack 'sources' are loaded, the additional subdirectories are created:

	   /src – contains the sources
	   /ant – contains the ant buildfile
	   /tests/validation – test files
           and some additional files required for IzPack are loaded in the root directory.
